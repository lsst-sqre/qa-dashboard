SQUASH

A Quality Control service developed by SQuaRE.

Purpose

SQUASH provides a minimal infrastructure for testing the LSST software stack against the Key Performance Metrics (KPMs) using precursor datasets. As a prototype, SQUASH can be used and extended by the scientists. The goal is to preserving the knowledge developed during LSST construction.

How to Use

Start by looking at the LSST Science Requirement metrics and check the existing features.
