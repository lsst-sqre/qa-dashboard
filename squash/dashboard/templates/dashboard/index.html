SQuaSH

A service developed by LSST SQuaRE.

Purpose

SQuaSH provides a minimal infrastructure for testing the LSST stack using precursor datasets. It can be used and extended by LSST DM to preserve the code and knowledge developed during LSST construction.


How to Use

The Monitor app shows a time series plot for each metric measured by validate_drp. Deviations in the metric measurements can be correlated with changes in the code. For some metrics drill down plots explain how the measurements are performed.
